# C-Programming-Codes


This repository contains my C programming programs completed as part of my first-year Mechanical Engineering coursework. The programs cover fundamental programming concepts, including variables, operators, loops, conditional statements, functions, arrays, strings, pointers, and file handling. The purpose of this repository is to strengthen my problem-solving skills and build a solid foundation in programming for engineering applications.

Topics Covered
Basic input and output
Conditional statements (if-else, switch)
Loops (for, while, do-while)
Functions and recursion
Arrays and strings
